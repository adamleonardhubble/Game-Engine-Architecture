Detailed Description

Class for OpenGL specific uniform buffers. Provides means for initialisation of OpenGL uniform buffers. Able to bind and unbind uniform buffer objects, bind shader uniform blocks to binding points. Able to retrieve uniform buffer object layouts also. Inherits from UniformBuffer class.
