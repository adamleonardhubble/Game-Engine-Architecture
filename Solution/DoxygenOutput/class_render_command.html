Detailed Description

Interface class for all render API render commands. Provides means for initialisation and creation of render commands. Able to clear depth buffer colour, set window clear colour and clear the window, set depth testing less for geometry fragments, set face culling of geometry facing away from the camera (back facing), clear colour writing buffers and enables alpha blending between geometry.
