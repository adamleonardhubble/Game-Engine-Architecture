Detailed Description

Class for orthographic specific cameras. Provides means for initialisation of orthographic camera properties. Able to reset the properties of a orthographic camera instance, update the view matrix of a orthographic camera and set and get the position and rotation of a orthographic camera instance. Inherits from the Camera class.
