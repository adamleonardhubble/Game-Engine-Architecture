Detailed Description

Class for OpenGL API specific materials. Provides means for initialisation of OpenGL materials. Able to set and get shaders, vertex array objects for geometry being rendered and unifrom per draw variable data which collectively compose a OpenGL API specific material. Inherits from Material class.
