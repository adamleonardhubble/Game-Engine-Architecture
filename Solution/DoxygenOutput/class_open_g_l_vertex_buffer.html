Detailed Description

Class for OpenGL specific vertex buffers. Provides means for initialisation of OpenGL vertex buffers. Able to bind and unbind vertex buffer objects to vertex attributes. Able to retrieve vertex buffer layout also. Inherits from VertexBuffer class.
