class  Application
 Fundamental class of the engine. A singleton patterned class which runs the game loop infinitely. Provides the means of initialising instances of windows and systems. Can also be used to create events and dispatch them. Aswell, renderer index buffers and vertex buffers can be set, whilst vertex arrays can be initialised. Shaders, textures and materials can be created and applied to geometry. Also, multiple camera types can be created for exploring a scene, or making alterations to the user interface. More...
